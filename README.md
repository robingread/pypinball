# Pi-Pinball

## Setup

Install dependencies:

```bash
./scripts/install-deps.sh
```

To develop, setup the virtualenv:

```bash
./scripts/setup-venv.sh
```

## Testing

To run the testing suite:

```bash
./scripts/ci-test.sh
```

## Documentation

To build the documentation, run:

```bash
source venv/bin/activate
./scripts/build-docs.sh
```
